# DINERI | Smart Finance Tracker

A modern, high-fidelity finance management workspace featuring voice commands, real-time analytics, and profile personalization.

## 🚀 Features

- **Voice Entry**: Log expenses and income hands-free using built-in speech recognition.
- **Dynamic Stats**: Interactive doughnut charts and category-wise spending efficiency.
- **Profile Personalization**: Upload custom avatars and sync profile data.
- **Live Balance**: Real-time automatic balance calculations.
- **Modern UI**: Obsidian Dark & Crystal Light modes with a smooth, responsive design.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla), Chart.js
- **Backend**: PHP 8.0+
- **Database**: MySQL (PDO)
- **Icons**: FontAwesome 6
- **Fonts**: Google Fonts (Outfit & Inter)

## 📦 Installation (Local via XAMPP)

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Yasine8710/track_it.git
   ```
2. **Setup Database**:
   - Create a database in phpMyAdmin named `track_it`.
   - Import the `database.sql` file.
3. **Configure Connection**:
   - Edit `includes/db.php` if you have custom MySQL credentials.
4. **Move to htdocs**:
   - Place the folder in `C:\xampp\htdocs\track_it`.
5. **Launch**:
   - Navigate to `http://localhost/track_it` in your browser.
